# Эмулятор машины Тьюринга

## Описание проекта

Программа представляет собой графический эмулятор машины Тьюринга.

Она позволяет задавать алфавиты, входную строку и таблицу переходов, а затем выполнять программу пошагово или автоматически.

---

## Основные возможности

- ввод алфавита ленты;
- ввод дополнительных символов;
- ввод входной строки;
- автоматическое добавление специального символа пустоты `_`;
- проверка корректности введённых данных;
- работа с таблицей переходов;
- выполнение программы по одному шагу;
- автоматический запуск программы;
- остановка выполнения;
- изменение скорости работы;
- сброс машины к начальному состоянию;
- добавление и удаление состояний;
- подсветка текущего состояния в таблице;
- графическое отображение ленты и головки.

---

## Интерфейс программы

Интерфейс программы состоит из нескольких основных блоков.

### 1. Алфавиты

В этом блоке пользователь задаёт:

- алфавит ленты;
- дополнительные символы.

Символ `_` является специальным символом пустоты.

Его нельзя вводить вручную в алфавит ленты или в дополнительные символы, так как программа добавляет его автоматически.

---

### 2. Входная строка

Пользователь вводит слово, которое будет записано на ленту перед началом выполнения программы.

Входная строка должна соответствовать следующим требованиям:

- не быть пустой;
- не содержать пробелов;
- состоять только из символов алфавита ленты;
- не содержать символ `_`.

---

### 3. Управление

Для управления работой машины используются кнопки:

| Кнопка | Назначение |
|---|---|
| Запустить | начинает автоматическое выполнение программы |
| Остановить | останавливает автоматическое выполнение |
| Один шаг | выполняет одну команду машины |
| Сброс | возвращает машину в начальное состояние |
| Быстрее | увеличивает скорость выполнения |
| Медленнее | уменьшает скорость выполнения |

---

### 4. Лента

Лента отображается графически.

На ней видно:

- отдельные клетки;
- символы, записанные в клетках;
- текущее положение головки.

При движении головки лента сдвигается, если головка подходит к краю области просмотра.

---

### 5. Таблица переходов

В таблице задаются команды машины Тьюринга.

Состояния имеют следующий вид:

```text
q0, q1, q2, ...

Пользователь может:

добавлять новые состояния;
удалять последнее состояние;
заполнять команды для каждого состояния и символа.
Формат команд

Команда в таблице записывается в формате:

новый_символ,направление,новое_состояние

Где:

новый_символ — символ, который будет записан в текущую клетку;
направление — направление движения головки;
новое_состояние — состояние, в которое перейдёт машина.

Возможные направления:

Обозначение	Значение
L	движение влево
R	движение вправо
S	остаться на месте
Примеры команд
b,R,q1

Записать символ b, сдвинуть головку вправо и перейти в состояние q1.

a,L,q2

Записать символ a, сдвинуть головку влево и перейти в состояние q2.

c,S,!

Записать символ c, оставить головку на месте и завершить выполнение программы.

Остановка программы

Символ ! используется для обозначения остановки машины.

Если после выполнения команды новое состояние равно !, программа завершает работу.
