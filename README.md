#  Эмулятор машины Тьюринга

> **Интерактивная графическая среда** для визуализации, проектирования и отладки алгоритмов классической машины Тьюринга.

Данный проект представляет собой мощный инструмент для изучения теории алгоритмов, позволяя пользователям вручную настраивать алфавиты, начальные данные и логику переходов с мгновенной визуализацией процесса.

---

##  Основные возможности

###  Гибкая настройка
* **Управление алфавитом:** Ввод основного алфавита ленты и дополнительных символов. 
* **Автоматизация:** Система сама добавляет специальный символ пустоты `_`.
* **Валидация:** Строгая проверка входной строки на соответствие алфавиту и отсутствие пробелов.

###  Режимы выполнения
* **Пошаговый (Step-by-step):** Идеально для отладки — выполнение ровно одной команды за раз.
* **Автоматический:** Запуск программы в непрерывном режиме.
* **Контроль скорости:** Динамическое ускорение или замедление работы машины прямо в процессе выполнения.
* **Сброс:** Мгновенный возврат к исходному состоянию и начальной строке.

###  Визуализация
* **Графическая лента:** Анимированное отображение ячеек и движения головки.
* **Умная камера:** Лента автоматически сдвигается, если головка выходит за пределы видимой области.
* **Интерактивная таблица:** Подсветка текущей выполняемой команды в таблице переходов.

---

##  Интерфейс программы

Программа разделена на логические модули:

1.  **Алфавиты:** Настройка доступных символов (ленты и вспомогательных).
2.  **Входная строка:** Поле для записи данных, с которых начнется обработка.
3.  **Панель управления:** Кнопки `Запустить`, `Остановить`, `Шаг`, `Сброс` и регуляторы скорости.
4.  **Визуальная лента:** Графическое поле с индексами ячеек и указателем головки.
5.  **Редактор таблицы:** Интерфейс для добавления/удаления состояний (`q0`, `q1`, ...) и ввода команд.

---

##  Формат команд

Команды в ячейках таблицы записываются через запятую в формате:
`Символ,Направление,Состояние`

| Компонент | Описание | Возможные значения |
| :--- | :--- | :--- |
| **Символ** | Что записать в текущую ячейку | Любой символ алфавита или `_` |
| **Направление** | Куда сдвинуть головку | `L` (влево), `R` (вправо), `S` (стоп/на месте) |
| **Состояние** | В какое состояние перейти | `q0`, `q1`, `q2`... или `!` для завершения |

### Примеры:
* `0,R,q1` — Записать `0`, шагнуть **Вправо**, перейти в состояние **q1**.
* `1,L,q0` — Записать `1`, шагнуть **Влево**, перейти в состояние **q0**.
* `_,S,!` — Записать пустоту, остаться **на месте**, **завершить** работу.
